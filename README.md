# calibracontrol
CalibraControl PRO é um sistema web de gestão de calibração de instrumentos industriais, desenvolvido para a GBR Componentes.
Funcionalidades:

Dashboard com KPIs de calibração (últimos 12/6/3 meses)
Cadastro e controle de equipamentos por setor, área e cliente
Histórico completo de calibrações realizadas
Gestão de certificados e avaliação técnica
Geração de etiquetas com QR Code para rastreabilidade
Alertas automáticos de vencimento (90 dias)
Relatórios gerenciais exportáveis em CSV
Controle de usuários com login e níveis de acesso
Log de auditoria completo

Tecnologias: HTML · CSS · JavaScript · Supabase (PostgreSQL) · Chart.js
Contexto: Ferramenta interna criada para substituir planilhas isoladas, centralizando o controle metrológico do parque de equipamentos.
